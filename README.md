📊 Snitch Fashion Sales Analysis Project

🔹 Overview

This project performs an end-to-end data analysis of a retail fashion sales dataset to uncover key business insights using Excel, SQL, Python, and Power BI.

🔹 Business Objectives

Identify top performing cities and products

Analyze monthly sales trends

Measure profit and discount impact

Build interactive business dashboard

🔹 Tools & Technologies

Excel – Data cleaning & pivot analysis

SQL (MySQL) – Business queries

Python – EDA & visualization

Power BI – Interactive dashboard

🔹 Project Workflow

Data Cleaning (Excel + Python)

SQL Business Queries

Python EDA & Visualization

Power BI Dashboard & Data Modeling

Business Insights & Recommendations

🔹 Key Insights

Ahmedabad & Bangalore generate highest revenue

Pune is lowest performing city

Dresses generate highest sales

Shoes produce highest profit

March 2025 shows peak growth
